Search Bar Visibility Test

Test Controls

Click the buttons to test the search visibility functionality.

Expected Behavior

  • When the page loads, the search bar and mobile toggle should be hidden (opacity: 0)
  • When the surf map opens, both elements should become visible (opacity: 0.9)
  • When the surf map closes, both elements should be hidden again

Manual Test Steps

  1. Open this page in a browser
  2. Verify the search elements are hidden (top-left corner)
  3. Click "Open Surf Map" button
  4. Verify the search elements become visible
  5. Click "Close Surf Map" button
  6. Verify the search elements are hidden again
